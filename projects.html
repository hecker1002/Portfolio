2D to 3D Reconstruction

Initialized a random 3D microstructure and then extracted feature maps for the actual 2D image . Now , in each iteration , coputed the loss for each slice ( using Gram Matrix = V. VT) and updated the voxel values iteratively using l-BFGS algorithm.

Trajectory Prediction

Used a CNN+RNN to iteratively find the future 10 steps of a pedestrian or vehicle in view of an autonomous driving vehicle.
